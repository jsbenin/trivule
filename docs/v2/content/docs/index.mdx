# Introduction

Trivule is a lightweight JavaScript library for form validation using simple HTML attributes. It allows you to define validation rules directly in your markup, making it easy to integrate without heavy JavaScript configurations.

## Quick Start

```html
<input type="text" data-tr-rules="required|int|min:18" name="age" />
```

Trivule parses the `data-tr-rules` attribute, validates on form submit or input events, and displays error messages.

## Features

- **Attribute-based rules**: Use `data-tr-rules` for validations like `required`, `email`, `min:18`, etc.
- **Zero dependencies**: Pure vanilla JavaScript.
- **Customizable**: Easily extend rules and error messages.
- **Framework-agnostic**: Works with any JS setup, including React, Vue, or plain HTML.

## Installation

```bash
npm install trivule
```

For more details, see the [Rules documentation](/docs/rules).
