# Why Trivule?

Trivule is an HTML attribute-based library that provides a simple API for validating forms directly in your markup, without leaving the HTML. It works with any JavaScript library but makes life significantly easier for developers using server-side frameworks like Laravel, Ruby on Rails, Phoenix, AdonisJS, Go with template engines, Fastify with Jinja, and others.

## Key Benefits

- **Effortless Integration**: Define validation rules using standard HTML attributes—no complex JavaScript setups required.
- **Universal Compatibility**: Functions independently or alongside any frontend framework (React, Vue, etc.) and backend templating systems.
- **Developer-Friendly for Server-Side**: Streamlines form handling in templated environments, reducing boilerplate code and improving productivity.
- **Lightweight and Zero Dependencies**: Pure vanilla JS, ensuring fast performance without bloat.
